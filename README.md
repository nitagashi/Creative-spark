# Character Codex

A modern writing reference library designed for storytellers, novelists, roleplayers, screenwriters, and creatives.

Character Codex helps writers collect, organise, and explore references for emotions, character traits, behaviours, injuries, body language, reactions, and countless other storytelling elements—all in one searchable library.

---

## Features

### Personal Writing Library

Create and manage your own collection of writing references.

Store entries for:

* Emotions
* Character traits
* Body language
* Behaviours
* Injuries
* Physical reactions
* Dialogue cues
* Personality archetypes
* Relationships
* Writing tips
* Worldbuilding references
* Custom categories

---

### Powerful Search

Quickly find references using:

* Keywords
* Categories
* Tags
* Titles
* Content search

No more digging through notebooks, documents, or bookmarks.

## Example Entry

### Nervousness

**Physical Signs**

* Fidgeting
* Avoiding eye contact
* Rapid blinking
* Tense posture
* Shallow breathing

**Behaviour**

* Speaking quickly
* Stumbling over words
* Repeating phrases
* Overexplaining

**Internal Experience**

* Racing thoughts
* Increased heartbeat
* Fear of judgement
* Difficulty concentrating

## Goal

Character Codex exists to make storytelling references easily accessible.

Instead of searching through dozens of websites, bookmarks, screenshots, and notes, writers can build a personal knowledge base that grows alongside their creativity.
